Logic: Conditional Statements

In JavaScript we have the following conditional statements:

  • Use if to specify a block of code to be executed, if a specified condition is true
  • Use else to specify a block of code to be executed, if the same condition is false
  • Use else if to specify a new condition to test, if the first condition is false
  • Use switch to specify many alternative blocks of code to be executed

For more examples and documentation, check out W3Schools

Basic If Statement

The most common "bread-and butter" logic statement is the IF statement 
if (condition) {
	code block
}

Branching Logic: if/else

To add a branch to the logic so that something happens if the condition is not met, add an 'else' statement to the end. the two outcomes are mutually exclusive, so only one of them will happen! 
if (condition) {
    code block if condition met
} else {
    code block if condition NOT met
}

The following will output "yes, it is Monday" if it is Monday, and will output "Nope, not Monday" if not.

Multiple Branches - If/Else If/Else

You can add as many else if() conditions as you need to after the first "if" to create multiple mutually-exclusive branches. else is optional, but must be the last statement if present. 
if (condition 1) {
	//code block if the condition 1 is met
} else if (condition 2) {
	//code block if the condition 1 is NOT met, but condition 2 is met
} else if (condition 3) {
	//code block if the condition 1 and 2 are NOT met, but condition 3 is met
} else {
	//code block if none of the conditions are met
}

Multiple Branches - Switch

The Switch statement allows a slightly more compact way to write multiple logic branches that evaluate the same expression. 
switch(expression){
	case 'value1' :
		//what to do if the expression matches value1
	break;
	case 'value2' :
		//what to do if the expression matches value2
	break;
	case 'value3' :
		//what to do if the expression matches value3
	break;
	default:
		//what to do if none of the cases match the expression

}
