Loops in JS

for Loop

FOR loops are good for doing something a set number of times, like generating 5 random numbers, or rolling a dice three times, or placing 6 images on the screen
const forEl = document.querySelector('#for .output');
//generate 5 random numbers
for (var i = 0; i <= 5; i++) {
	let rand = Math.round( Math.random() * 10 ) ;
	forEl.insertAdjacentHTML('beforeend', `The for loop is on iteration ${i}, The random number is ${rand}<br>`);			
}

forEach Loop

for each loops are specifically for looping through each item in an array. Here we have a list of players and we want to show a list item for each one: 
const forEachEl = document.querySelector('#for-each ol.output');
//show all the names of all the players in the list
allPlayers = ['Alfonso', 'Georgia', 'Barry', 'Soleil', 'Terry', 'Cynthia'];

allPlayers.forEach(function(player) {
   forEachEl.insertAdjacentHTML('beforeend', `<li>${player}</li>`);	
});

    While Loop

    While loops are for conditions that will eventually be met, like a counter that will increase until it reaches a limit. This example adds a random number to an amount until the amount reaches 50. A while loop will do nothing if the condition is not met. if the amount starts higher than 50, nothing will output! 
    const whileEl = document.querySelector('#while .output');
////get a random number between 0-100
let amount = Math.round(Math.random() * 100);
while(amount <= 50){
	//get a random amount from 0-10
	rand = Math.round(Math.random() * 10);
	whileEl.insertAdjacentHTML('beforeend', `The amount is currently ${amount} <br>`);
	amount += rand;	
}

    Do While Loop

    Do While loops are just like While loops, except they always run at least once, even if the condition is not met. if the amount starts higher than 50, one line will output. 
    const doWhileEl = document.querySelector('#do-while .output');
amount = Math.round(Math.random() * 100);
do{
	//get a random amount from 0-10
	rand = Math.round(Math.random() * 10);
	doWhileEl.insertAdjacentHTML('beforeend', `The amount is currently ${amount} <br>`);
	amount += rand;	
}while(amount <= 50);
